# Build and Fix

TypeScript とビルドエラーを段階的に修正します:

1. ビルド実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重大度順にソート

3. 各エラーについて:
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解消されたか確認

4. 停止条件:
   - 修正により新しいエラーが発生した場合
   - 3回試行しても同じエラーが続く場合
   - ユーザーが一時停止を要求した場合

5. サマリーを表示:
   - 修正されたエラー
   - 残っているエラー
   - 新たに発生したエラー

安全のため、一度に1つのエラーを修正してください！
