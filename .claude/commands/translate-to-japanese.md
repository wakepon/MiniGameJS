# 日本語翻訳コマンド

指定されたファイルを日本語に翻訳して書き換えます。

## 対象ファイル

$ARGUMENTS

## 翻訳ルール

1. **プログラミング用語は英語のまま残す**
   - 変数名、関数名、クラス名
   - Agent, Task, Hook などの技術用語
   - API, REST, GraphQL などの略語
   - Git, Docker, Node.js などのツール名

2. **日本語に翻訳する部分**
   - 説明文、コメント
   - 見出し（技術用語を含む場合は英語を残す）
   - 表のヘッダー（「Purpose」→「目的」など）
   - 例文の説明部分

3. **フォーマットを維持**
   - Markdownの構造を保持
   - コードブロックの内容はそのまま
   - テーブルのレイアウトを維持

## 実行手順

1. 対象ファイルを読み込む
2. 上記ルールに従って日本語に翻訳
3. ファイルを上書き保存
4. 翻訳結果を確認
